# Sales Performance & Business Insights Dashboard

## Project Overview
This project focuses on analyzing sales data to identify business
trends, regional performance, top products, and loss-making areas.
The goal is to support data-driven decision-making.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Power BI
- Excel

## Workflow
1. Cleaned and prepared raw sales data using Python and Excel
2. Imported cleaned data into MySQL database
3. Performed SQL-based business analysis
4. Built interactive Power BI dashboards

## Future Enhancements
- Automate data refresh using scheduled pipelines
- Add forecasting models for sales prediction
- Integrate real-time data sources
- Perform customer segmentation for targeted marketing

## Key Business Insights
- West region performs best in both sales and profit
- Furniture category has high sales but lower profitability
- Deep discounts negatively impact profit margins
- Sales peak during festive months (Novâ€“Dec)

## KPIs Defined
- Total Sales: Total revenue generated from all orders
- Total Profit: Net profit after discounts and costs
- Total Orders: Number of unique customer orders

## Key Insights
- Identified top-performing regions and categories
- Analyzed monthly sales trends and seasonality
- Found loss-making products requiring optimization
- Created KPI dashboards for quick business overview

## Business Questions Answered
1. Which regions generate the highest sales and profit?
2. Which product categories contribute most to revenue?
3. What are the monthly sales trends and seasonality?
4. Which products are causing losses?
5. How does discount impact profitability?
